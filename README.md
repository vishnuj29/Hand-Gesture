# ✨ Hand Gesture Drawing App ✋✍️

An interactive web app that lets users draw on screen using **hand movement (webcam)** and **keyboard controls**.

---

## 🚀 Features

- ✋ Hand tracking using webcam  
- ✍️ Draw using index finger  
- ⌨️ Hold **Shift** to draw  
- ⌨️ Press **Space** to clear screen  
- ✨ Smooth glowing drawing  

---

## 🎮 Controls

| Action | Key |
|--------|-----|
| Move Cursor | Move finger |
| Draw | Hold Shift |
| Stop | Release Shift |
| Clear | Press Space |

---

## 🧠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- MediaPipe Hands  
- Canvas API  

---

## ⚙️ How to Run

1. Clone repo:
```bash
git clone https://github.com/your-username/hand-gesture-drawing.git
cd hand-gesture-drawing
```

2. Run using Live Server (VS Code)  
OR  
```bash
npx serve
```
